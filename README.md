## 📌 Sobre o projeto

O **Tá Pago?** é um aplicativo web simples que ajuda usuários a controlarem suas contas do dia a dia de forma prática, visual e intuitiva — sem necessidade de cadastro ou integração com bancos.

Desenvolvido durante um **hackathon** com foco em soluções para o setor financeiro.

---

## 🎯 Problema

Muitas pessoas enfrentam dificuldades para:

- Lembrar se já pagaram uma conta;
- Acompanhar datas de vencimento;
- Evitar atrasos e juros desnecessários;
- Organizar as finanças de forma simples e rápida.

---

## 💡 Solução

O **Tá Pago?** permite que o usuário:

- Cadastre suas contas manualmente;
- Visualize quais estão **pendentes**, **pagas** ou **atrasadas**;
- Marque contas como pagas com um clique;
- Acompanhe vencimentos de forma clara;
- Tenha um **resumo financeiro** direto na tela.

---

## 🧠 Funcionalidades

| Funcionalidade | Descrição |
|---|---|
| ➕ Adicionar conta | Cadastre nome, valor, vencimento e forma de pagamento |
| 📋 Listar contas | Visualize todas as contas cadastradas em um só lugar |
| ✔️ Marcar como paga | Confirme pagamentos e desfaça se necessário |
| 🗑 Excluir conta | Remova contas que não precisa mais acompanhar |
| 📊 Resumo financeiro | Veja o total pendente, pago e atrasado |
| 🎯 Filtros | Filtre por: todas, pendentes ou pagas |
| 💾 Armazenamento local | Seus dados ficam salvos no próprio navegador |
